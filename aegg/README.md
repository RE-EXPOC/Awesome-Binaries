AEGG
----

AEGG is my simple automatic exploit generation tool. Now only support x86 binary.

Usage
-----

1. vi `vul.c`:
    ```
    #include <stdio.h>
    #include <unistd.h>
    #include <string.h>

    char buf[100];

    int sample_func() {
        char name[10] = {0};
        read(0, buf, 307);
        strcpy(name, buf);
        printf("input: %s\n", name);
    }

    int main(void)
    {
        printf("Running...\n");
        sample_func();
        printf("Done.\n");
    }
    ```
2. complie `vul`:
    ```
    gcc vul2.c -o vul2 -m32 -g -z execstack
    ```
3. vi `my_aegg.py`:
    ```
    from aegg import AEGG

    binary = './vul'
    gg = AEGG(binary)

    # generating payload!
    gg.hack()

    print repr(gg.payloads[0].content)
    gg.save()
    ```
